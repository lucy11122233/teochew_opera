# 妈妈的潮剧大剧院 (Teochew Opera Home)

一个专为潮汕长辈设计的超简易潮剧视频聚合网页应用。

## 项目特色

### 🎭 核心定位
- **零学习成本**：超大按钮、超大字体、极简操作
- **乡音亲切**：专为潮汕长辈定制的潮剧内容
- **传统美学**：采用潮汕传统红金配色，营造戏院氛围

### ✨ 主要功能

#### 1. 视频浏览与播放
- 经典潮剧剧目展示（苏六娘、陈三五娘等）
- Bilibili视频集成，支持全屏播放
- 智能分类：才子佳人、经典苦情、热闹武戏

#### 2. 语音搜索
- 长按语音按钮，说出想看的剧目
- 支持自然语言搜索："我想看姚璇秋"
- 智能识别并展示搜索结果

#### 3. 个人收藏
- 一键收藏喜爱的剧目
- 收藏列表独立展示
- 快速访问喜爱的内容

#### 4. 家人留言板
- 家人可以远程发送关怀留言
- 留言显示在首页显著位置
- 支持文字留言，传递温暖

#### 5. 智能推荐
- 显示当前农历日期
- 节日自动推荐相关剧目
- 节气提醒与祝福

#### 6. 远程管理
- 管理员后台（密码保护）
- 添加新视频链接
- 管理家人留言

## 技术特性

- **前端框架**：React.js 18

- 符合WCAG无障碍标准

- 减少动画模式支持

## 安装与运行

### 环境要求
- Node.js 16+
- npm 或 yarn

### 安装步骤

1. 克隆项目到本地
```bash
git clone [项目地址]
cd teochew-opera-home
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm start
```

4. 打开浏览器访问 `http://localhost:3000`

### 构建生产版本
```bash
npm run build
```

## 使用方法

### 日常使用

#### 观看潮剧
1. 打开应用，直接显示推荐内容
2. 点击剧目封面即可播放
3. 支持全屏观看，自动隐藏控制栏

#### 搜索剧目
1. 在搜索框输入剧目名称或演员
2. 或点击语音按钮，说出想看的剧目
3. 系统会自动筛选并展示结果

#### 收藏剧目
1. 在剧目卡片点击爱心按钮即可收藏
2. 在"我的收藏"中查看所有收藏的剧目
3. 再次点击可取消收藏

### 管理功能

#### 访问管理后台
1. 在网址后加上 `/admin` 访问管理页面
2. 输入管理员密码（默认为：mama123）
3. 进入后台管理界面

#### 添加新视频
1. 在管理后台选择"添加视频"
2. 填写剧目信息：
   - 剧目标题
   - 分类选择
   - B站BV号
   - 封面图片URL
   - 剧情简介
   - 时长、主演、年份
   - 标签（用逗号分隔）
3. 点击"添加视频"保存

#### 管理家人留言
1. 选择"添加留言"
2. 输入留言内容和署名
3. 点击"发送留言"
4. 留言会立即显示在用户首页

```json

  "id": "unique_id",
  "title": "苏六娘",
  "category": "才子佳人",
  "bvid": "BV1Zs411t7dL",
  "thumbnail": "封面图片URL",
  "description": "剧情简介",
  "duration": "02:45:30",
  "tags": ["姚璇秋", "经典", "爱情"],
  "performer": "姚璇秋",
  "year": "1958"
}

  "date": "2026/1/6"
}
```

## 自定义配置

### 修改配色方案
在 `src/styles.css` 中修改CSS变量：
```css
--primary-color: #B22222;    /* 宫廷红 */
--accent-color: #FFD700;     /* 戏冠金 */
--bg-color: #1A0A05;         /* 剧场暗红 */
```

### 添加新分类
在 `src/data/operas.json` 中添加新分类：
```json
{
  "id": "new_category",
  "name": "新分类名称",
  "icon": "分类图标"
}
```

### 修改管理员密码
在 `src/components/Admin.js` 中修改：
```javascript
const ADMIN_PASSWORD = 'your_new_password';
```

## 部署指南

### 部署到 Vercel
1. 连接GitHub仓库到Vercel
2. 选择项目并配置构建设置
3. 点击部署按钮

### 部署到 GitHub Pages
1. 构建项目：`npm run build`
2. 将 `build` 文件夹内容推送到 `gh-pages` 分支
3. 在GitHub Pages设置中启用页面

### 部署到 Netlify
1. 拖拽 `build` 文件夹到Netlify
2. 或连接GitHub仓库自动部署
