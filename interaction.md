# 妈妈的潮剧大剧院 - 交互设计方案

## 核心交互功能

### 1. 超大导航按钮
- **看大戏**：经典潮剧全本
- **听唱段**：精选唱段集锦
- **我的收藏**：个人收藏夹
- **家人留言**：查看家人留言

### 2. 视频浏览与播放
- 网格布局展示视频缩略图，每个视频卡片包含：
  - 大尺寸封面图（16:9比例）
  - 剧名（24px+字体）
  - 分类标签
- 点击视频卡片直接全屏播放
- 支持滑动浏览更多视频

### 3. 智能语音搜索
- 长按语音按钮激活语音识别
- 支持自然语言搜索：
  - "我想看苏六娘"
  - "播放姚璇秋的戏"
  - "找喜庆的剧目"
- 语音搜索结果自动展示

### 4. 远程管理功能
- 管理员界面（密码保护）
  - 添加B站视频链接
  - 编辑视频分类
  - 管理收藏列表
  - 发布家人留言

### 5. 智能推荐系统
- 根据节日自动推荐：
  - 春节：播放贺岁戏
  - 元宵：播放喜庆剧目
  - 中秋：播放团圆主题戏
- 显示当前农历日期和节气

### 6. 家人留言板
- 文字留言：简短祝福语
- 语音留言：录制关怀话语
- 留言显示在首页显著位置

## 用户体验流程

### 首次访问
1. 显示欢迎页面和大字操作指引
2. 自动播放推荐剧目
3. 引导用户尝试语音搜索

### 日常使用
1. 打开应用直接显示推荐内容
2. 一键播放，无需复杂操作
3. 收藏功能简单易用

### 远程更新
1. 家人通过管理后台添加新戏
2. 母亲端自动同步更新
3. 新内容会有明显标识

## 移动端优化

- 所有按钮最小高度80px，方便触摸
- 字体大小24px起，确保清晰可读
- 支持手势滑动浏览
- 语音按钮固定在右下角，方便随时使用
- 全屏播放时自动隐藏控制栏

## 无障碍设计

- 高对比度色彩方案
- 大字体显示
- 语音反馈确认操作
- 简化操作流程，减少学习成本
