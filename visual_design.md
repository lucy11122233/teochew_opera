# 妈妈的潮剧大剧院 - 视觉设计方案

## 1. 设计理念

### 整体视觉方向

- 以"戏台光影"为核心，营造温暖、热闹且亲切的怀旧氛围。设计灵感源自老式戏院的布景与灯光，将用户界面转化为一个充满情感与记忆的家庭剧场。整体设计旨在通过强烈的传统色彩和清晰的视觉层级，为长辈用户提供零学习成本的直观体验。

### 内容密度

- 中等密度。专为长辈设计，采用大字体、大按钮和充足的留白，确保视觉清晰、操作简便。每页仅呈现2-6个核心选项，避免信息过载，优先保证易用性和舒适度。

## 2. 色彩系统

**主色调**

- 背景色：#1A0A05（深邃的剧场暗红，营造戏院氛围）
- 主色：#B22222（宫廷红，传统潮剧戏服的主色调）
- 辅助色：#8B0000（暗红色，用于阴影和次要元素）
- 强调色：#FFD700（戏冠金，象征荣耀与喜庆）

**文字颜色**

- 深色文字（用于浅色背景）：#2E2E2E
- 浅色文字（用于深色背景）：#FFF8DC（温暖的米白色，如戏台灯光）

## 3. 字体设计

**标题字体**

- 中文字体
  - Noto Serif SC（一款典雅的宋体，具有传统书卷气，适合营造潮剧的古典韵味）
- 英文字体
  - Oranienbaum（一款带有古典气质的现代衬线体，线条优雅，富有戏剧感）

**正文字体**

- 中文字体
  - Noto Sans SC（中性、清晰的黑体，确保长辈用户能够轻松阅读屏幕上的文字）
- 英文字体
  - Quattrocento Sans（一款经典无衬线体，形态温和，可读性强，适合大段文字阅读）

## 4. 图像风格

- 图像应具有温暖、怀旧的质感。视频封面可采用柔和的暗角和金色滤镜处理，增强戏台光影效果。UI图标可选用简洁的线性风格，融入传统纹样如祥云、戏帽等元素，但需保持高度简洁和清晰，避免干扰主要功能。
